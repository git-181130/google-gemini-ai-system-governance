# Contributing Guidelines

Thank you for your interest in this project.

This repository documents a completed AI product evaluation, operations, and governance framework.  
It is published primarily as a reference and portfolio artifact.

## Scope of Contributions

At this time:
- Feature additions are not accepted
- Changes to evaluation outcomes or case studies are not accepted
- Framework definitions are considered frozen

## Acceptable Contributions

The following may be considered:
- Typographical fixes
- Clarifications that do not alter meaning
- Documentation improvements that preserve scope and intent

## How to Propose Changes

If you believe a correction is necessary:
1. Open an issue describing the concern
2. Provide a clear rationale
3. Reference the specific document and section

All contributions are reviewed for scope alignment and governance consistency.

## Code of Conduct

All interactions should remain professional and respectful.
